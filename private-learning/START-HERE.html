Modo Iniciante

Use sem medo

Este painel foi feito para voce preencher sem conhecer o repo inteiro. O objetivo nao e "mexer certo em tudo". E so fechar uma sessao simples de trabalho sem se perder.

Regra central

Nada daqui toca o repo principal `OLMO` sem sua autorizacao explicita.

Como pensar

  • Se tem prazo, vai para Deadline.
  • Se voce precisa aprender, vai para Study.
  • Se ainda e pergunta aberta, vai para Experiment.
  • Se ja funcionou localmente, pode virar Candidate.

Ritmo

  • `30 min`: capturar, rotear, mover 1 item.
  • `1 h`: capturar, rotear, estudar 1 tema e fechar log.
